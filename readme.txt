== Description ==

= MAIN FEATURES OF THE WP-OBS SIDEBAR WIDGET =

This Plugin is a sidebar widget for websites who run the WP-OBS PRO <strong>[The Online Booking System]</strong> (You can download a free trial of the WP-OBS PRO  <a href=" http://wp-online-booking-system.com/" title="The WP-OBS PRO">here</a>) . You must have the <a href=" http://wp-online-booking-system.com/order" title="The WP-OBS PRO">WP-OBS PRO</a> installed for it to work.

The widget will enable you to add any or all services from your WP-OBS PRO to your website sidebar. You can then change the colours and font size of the links as you find suitable for your website.


You must run Wordpress version <strong>3.5.2</strong> or higher

= The WP-OBS [Online Booking Calendar] is great for: =

* Multilingual customised booking/appointment form (front end)

* Fully customised email alerts templates

* PayPal & Mailchimp Add-on available.

== Installation ==

1. Upload the zip file via your plugin upload section
1. Activate the plugin
1. Navigate to the widget area
1. Set up the widget using the settings on the widget
